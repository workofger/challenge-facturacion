# Challenge: Portal de Facturación

## Descripción

Portal web donde proveedores pueden subir, gestionar y dar seguimiento a sus facturas CFDI. Incluye un panel administrativo y un portal de usuario.

El frontend está funcional con datos mock. Tu reto es construir el backend que lo haga funcionar con datos reales.

## Tech Stack (Frontend)

| Capa | Tecnología |
|------|-----------|
| Framework | React 18 + TypeScript |
| Build | Vite 5 |
| Estilos | Tailwind CSS 3 |
| Routing | React Router 6 |

## Inicio Rápido

```bash
npm install
npm run dev
```

La aplicación correrá en `http://localhost:3000`. Navega por las páginas para entender la funcionalidad.
